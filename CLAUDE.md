# Claude Guidance

Read `AGENTS.md` first. It is the canonical TraderCat project brief and implementation guide for Claude, Codex, and any other agent working in this repo.

TraderCat is a **mobile-first web app** (Next.js on Vercel, Neon Postgres, Inngest, Stripe) — a **trading journal + AI trade review** product (Tradezella-style) with portfolio intelligence as a secondary lens. It is **not** a native app; ignore any older React Native / Firebase / RevenueCat guidance.

Do not duplicate or drift from `AGENTS.md`. If guidance needs to change, update `AGENTS.md`.
